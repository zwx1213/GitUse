### Git 版本管理

#### 1、git init

        git仓库初始化

        .git非常重要，存放记录

    

#### 2、git log

        查看git提交日志

    

#### 3、git status

        查看git是否处于提交缓冲状态

    

#### 4、git add  <file>

        将文件提交到缓冲区

    

#### 5、git commit -m "<提交附带的提示标题信息>"

        给提交操作赋予名称

    

#### 6、git branch -M <分支名称>

        给git操作建立新分支，并赋予名称

    

#### 7、git remote add origin   <url>

        添加远程仓库

    　　要添加一个新的远程仓库,可以指定一个简单的名字,以便将来引用,运行

```git命令
git remote add [shortname] [url]
```
